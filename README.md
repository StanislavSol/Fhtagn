# Генератор числового треугольника

## Как использовать

1. Скопируйте код в файл `triangle.php`
2. Запустите в командной строке:

```bash
php triangle.php <число>
```

### Примеры:

```bash
php triangle.php 9
```
Выведет:
```
      1
    2 3 4
  5 6 7 8 9
```

```bash
php triangle.php 12
```
Выведет:
```
Невозможно построить треугольник
```

## Требования
- PHP 7.4+
- Командная строка
